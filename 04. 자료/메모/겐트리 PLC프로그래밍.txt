PLC 프로그래밍

--------
원점 복귀 기동

[ORG 1 0]
[ORG OP1 OP2]
OP1 : 위치결정 모듈이 장착된 베이스 및
슬롯 번호를 작성하게 되어 있음
OP2 : 명령을 내릴 축을 성정하는 부분
축을 1 개만 사용하며 배선을 X 축을
하였기에 0번으로 설정
--------
직접 기동

[DST 1 0 1000 200 0 0 h0012]
[DST OP1 OP2 OP3 OP4 OP5 OP6 OP7]
OP1~OP2는 ORG 명령과 동일하게 설정을 
한 후 OP3 에서 이동할 위치를 설정(1000 = 1mm)
3만당 약 1초(이동시간)
OP4 에서 DST 명령을 실행하여 위치이동 시
이동속도를 정할 수 있음
OP5~OP6 은 0 으로 설정하며 자세한 내용은 
사용설명서를 참고
OP7 은 워드 단위인 제어 워드롤 비트에 
따른 제어 방법을 설정할수 있음(0 = 절대, 16 = 상대)
